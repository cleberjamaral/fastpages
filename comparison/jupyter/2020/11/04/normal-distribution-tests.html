Normal distribution tests

Using a range of normal distribution test functions

Nov 4, 2020 • Cleber Jorge Amaral • 2 min read

comparison   jupyter

Tests on a true normal distribution

Using the tests in a normal distribution in which it is expected to get true from results. Notice that since the data is randomly generated false-negative results may happen.

Using D'Agostino and Pearson's (DP) test, it is a normal distribution ( 0.073 > 0.05 ). DP-test = 5.227
Using Lilliefors test, it is a normal distribution ( 0.222 > 0.05 ). Lilliefors-test = 0.02
Using Shapiro-Wilk (SW) test, it is a normal distribution ( 0.089 > 0.05 ). SW-test = 0.997
Using Anderson-Darling (AD) test, it is a normal distribution ( 0.784 > 0.514 ).

 4 in 4 tests says it is likely to be a normal distribution ( alpha = 0.05 )

Testing an approximate normal distribution

This next distribution is closelly normal. In order to get less false-negative results alpha value is being set to a lower value.

Using D'Agostino and Pearson's (DP) test, it is NOT a normal distribution ( 0.018 NOT > 0.05 ). DP-test = 8.042
Using Lilliefors test, it is a normal distribution ( 0.518 > 0.05 ). Lilliefors-test = 0.02
Using Shapiro-Wilk (SW) test, it is a normal distribution ( 0.089 > 0.05 ). SW-test = 0.997
Using Anderson-Darling (AD) test, it is a normal distribution ( 0.784 > 0.503 ).

 3 in 4 tests says it is likely to be a normal distribution ( alpha = 0.05 )

Testing against an exponential distribution

This next distribution is not normal at all, no true results are expected

Using D'Agostino and Pearson's (DP) test, it is NOT a normal distribution ( 0.0 NOT > 0.05 ). DP-test = 421.117
Using Lilliefors test, it is NOT a normal distribution ( 0.0 NOT > 0.05 ). Lilliefors-test = 0.17
Using Shapiro-Wilk (SW) test, it is NOT a normal distribution ( 0.0 NOT > 0.05 ). SW-test = 0.798
Using Anderson-Darling (AD) test, it is NOT a normal distribution ( 0.784 NOT > 53.065 ).

 0 in 4 tests says it is likely to be a normal distribution ( alpha = 0.05 )
