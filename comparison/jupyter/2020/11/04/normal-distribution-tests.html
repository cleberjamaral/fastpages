Normal distribution tests

Using a range of normal distribution test functions

Nov 4, 2020 • Cleber Jorge Amaral • 2 min read

comparison   jupyter

Tests on a true normal distribution

Using the tests in a normal distribution in which it is expected to get true from results. Notice that since the data is randomly generated false-negative results may happen.

Using D'Agostino and Pearson's (DP) test, it is a normal distribution ( 0.927 > 0.05 ). DP-test = 0.151
Using Lilliefors test, it is a normal distribution ( 0.721 > 0.05 ). Lilliefors-test = 0.02
Using Shapiro-Wilk (SW) test, it is a normal distribution ( 0.59 > 0.05 ). SW-test = 0.999
Using Anderson-Darling (AD) test, it is a normal distribution ( 0.784 > 0.331 ).

 4 in 4 tests says it is likely to be a normal distribution ( alpha = 0.05 )

Testing an approximate normal distribution

This next distribution is closelly normal. In order to get less false-negative results alpha value is being set to a lower value.

Using D'Agostino and Pearson's (DP) test, it is a normal distribution ( 0.608 > 0.05 ). DP-test = 0.996
Using Lilliefors test, it is a normal distribution ( 0.979 > 0.05 ). Lilliefors-test = 0.01
Using Shapiro-Wilk (SW) test, it is a normal distribution ( 0.816 > 0.05 ). SW-test = 0.999
Using Anderson-Darling (AD) test, it is a normal distribution ( 0.784 > 0.168 ).

 4 in 4 tests says it is likely to be a normal distribution ( alpha = 0.05 )

Testing against an exponential distribution

This next distribution is not normal at all, no true results are expected

Using D'Agostino and Pearson's (DP) test, it is NOT a normal distribution ( 0.0 NOT > 0.05 ). DP-test = 417.669
Using Lilliefors test, it is NOT a normal distribution ( 0.0 NOT > 0.05 ). Lilliefors-test = 0.17
Using Shapiro-Wilk (SW) test, it is NOT a normal distribution ( 0.0 NOT > 0.05 ). SW-test = 0.806
Using Anderson-Darling (AD) test, it is NOT a normal distribution ( 0.784 NOT > 52.939 ).

 0 in 4 tests says it is likely to be a normal distribution ( alpha = 0.05 )
