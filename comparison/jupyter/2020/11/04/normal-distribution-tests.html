### Statitistical normality check

Using D'Agostino and Pearson's (DP) test, it is a normal distribution ( 0.259 > 0.05 ). DP-test = 2.703
Using Lilliefors test, it is a normal distribution ( 0.431 > 0.05 ). Lilliefors-test = 0.02
Using Shapiro-Wilk (SW) test, it is a normal distribution ( 0.55 > 0.05 ). SW-test = 0.998
Using Anderson-Darling (AD) test, it is a normal distribution ( 0.784 > 0.331 ).

 4 in 4 tests says it is likely to be a normal distribution ( alpha = 0.05 )

Testing an approximate normal distribution

This next distribution is closelly normal. In order to get less false-negative results alpha value is being set to a lower value.
